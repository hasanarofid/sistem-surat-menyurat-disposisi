# Sistem Surat Menyurat & Disposisi Digital (E-Office)

Sistem manajemen persuratan modern yang dibangun dengan **Laravel 11** dan **Vue 3**. Sistem ini dirancang untuk mendigitalkan alur surat masuk, surat keluar, dan disposisi instruksi dengan antarmuka yang premium, responsif, dan diduking oleh fitur pencarian data yang canggih.

## ✨ Fitur Utama
- **📊 Dashboard Statistik**: Visualisasi data surat masuk dan keluar secara real-time dengan grafikinteraktif.
- **📥 Manajemen Surat Masuk**: Pencatatan surat masuk, pengarsipan digital, dan delegasi disposisi.
- **📤 Manajemen Surat Keluar**: Penomoran otomatis dan pencatatan tujuan surat resmi instansi.
- **🔄 Alur Disposisi Timeline**: Pelacakan instruksi pimpinan secara kronologis dengan tampilan timeline premium.
- **📂 Arsip Digital**: Penyimpanan dokumen permanen dengan filter kategori dan pencarian cepat.
- **👥 Manajemen Pengguna (RBAC)**: Pengaturan hak akses bertingkat (Admin, Pimpinan, Staff).
- **🌓 Theme Support**: Dukungan Mode Gelap (Dark Mode) dan Terang secara persisten.

## 🛠️ Teknologi Utama
- **Backend**: Laravel 11, Sanctum API (Authentication), MySQL.
- **Frontend**: Vue 3, Vite, Tailwind CSS, Pinia (State Management), Lucide Icons.
- **Development**: AI Driven Development (Modern & Clean Code).

## 🚀 Instalasi Cepat
1. Clone repositori ini.
2. Jalankan `composer install` dan `npm install`.
3. Salin `.env.example` ke `.env` dan sesuaikan database.
4. Jalankan migrasi dan seeding: `php artisan migrate:fresh --seed` (Menghasilkan 200 data dummy).
5. Jalankan server: `php artisan serve` & `npm run dev`.
